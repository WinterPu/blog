Kitchen Crisis

It's a tower-defense game

4 People Team in NDHU

Game Bacis

It's a tower-defense game made by Unity.

The game goal is: you need to control the number of escaped food is less than required in each level. It means you need to defeat the food as much as possible. You also need to achive hidden requests to lock the special recipe.

Game Features
  • Multiple recipe collection
  • Change pot temperature to change its attack
  • Different evolution ways for pots
